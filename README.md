# Knowledge Library Management System

A desktop Library Management System developed using **Java** and **JavaFX**. The application provides an easy-to-use interface for managing library operations, including user login, book browsing, borrowing, purchasing, and invoice generation.

---

## Features

* User Login System
* Book Management
* Borrow Books
* Purchase Books
* Invoice Generation
* User-Friendly JavaFX Interface
* Input Validation and Error Handling

---

## Technologies Used

* Java
* JavaFX
* Apache NetBeans
* Ant Build System
* Text Files (.txt) for Data Storage

## Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/Project__Knowlegde.git
```

2. Open the project using Apache NetBeans.

3. Make sure Java and JavaFX are installed and configured.

4. Build and run the project.

---

## How to Use

1. Launch the application.
2. Log in using a valid student account.
3. Browse available books.
4. Borrow or purchase books.
5. View the generated invoice after completing a purchase.
